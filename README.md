### Example of use
```
# With python
git clone https://github.com/mhaut/pycircularstats/
cd pycircularstats
python setup.py install --user


# With conda
git clone https://github.com/mhaut/pycircularstats/
conda env create -f enviroment.yml
```

### Run examples
```
cd examples
python example.py
```
